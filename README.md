# Quickbooks-Data-Dump
Exporting QBO data to excel for reporting

## End-User License Agreement (EULA)

This qbo python script is for internal use by Steam Turbine Services employees only.
It is used to export invoices, purchase orders, and other financial data
for internal reporting. External distribution is not allowed.


## Privacy Policy

### Data Access
This application accesses QuickBooks Online financial data such as invoices,
purchase orders, vendors, and related accounting records.

### Data Usage
The retrieved data is used solely for internal reporting, analysis,
and operational business processes within Steam Turbine Services.

### Data Sharing
No QuickBooks data retrieved by this application is shared with
third parties.

### Data Storage
Any exported data is stored only within company-controlled systems
and is protected by internal access controls.
